１目的

この計画は、神戸市地域防災計画、神戸市水防計画及び神戸市消防局風水害対応計画に基づき、西区における風水害に係る西消防署の処理すべき業務に関して必要な事項を定め、迅速かつ適切な水防活動の実施を図ることを目的とする。

備考

  1. 防災センターで研修中の職員は、センター長が必要と認める場合を除き、防災指令第２号までの招集を除外する。
  2. 救急救命士の資格に関する病院研修中の職員は、救急部長が必要と認める場合を除き、防災指令第２号までの招集を除外する。ただし、防災指令第３号発令時は、各所属等へ参集する。
  3. 再任用職員は防災指令第２号までの招集を除外する。
  4. 西97小隊に関しては、防災指令第２号まで招集を免除とする。

  • ２号発令時：総務査察課＝係長２名・係員５名／消防防災課＝消防団係２名（消防団に係る業務）、救急係長・地域防災担当者（指揮活動班業務）
  • ３号発令時：全員
招集の方法

職員

  • 職員の招集は、別途定める西消防署緊急連絡網により行う。職員は、防災指令の発令前であっても発令の可能性があると判断した場合は情報収集に努め、発令に備える。
  • 交替制勤務者は、防災指令第２号発令の際、各係１班・２班と交互に招集する。なお、防災指令が長期化し非番が連続する場合は、班を入れ替える。
  • 日勤者は、２班交替として事案の都度班を入れ替える。防災指令が長期化する場合は、８時45分に班を交替する。
  • 参集しなくてはならない班に旅行届等で参集できない職員がいる場合は、違う班から参集人員を確保する。この場合の参集者は、災害待機手当での処理とする。

消防団員

  • 署長は、警防規程第53条第３項第４号に基づき、消防団長に消防団を招集し、警備を実施するよう依頼するものとする。
  • 消防団長は、防災指令第１号発令時、消防団員を招集して各支団本部を各区役所出張所内に設置する。招集の方法は、支団ごとに予め定められた連絡計画に基づく。なお、消防団の活動等に関することは消防団水防計画書に定める。

５署本部の運用

別紙１－３ 西消防署本部業務分担表参照。別紙１－４ 水防時体制業務分担表（必要事項記入用）を活用すること。

（１）指揮活動班の運用

当務指揮隊と参集指揮隊の任務（当務指揮が優先する原則）

防災指令第２号及び第３号の発令下において、以下の災害発生状況であれば参集指揮隊も指揮所運営を行う。

  • 当務指揮隊の出動
  • 管内の災害対応件数が５件以上
  • 避難指示事案の発生
  • 河川水位の氾濫注意に達したとき
  • 指揮活動班のローテーション
  • 大隊長命による指示
（２）情報班の運用

情報班にあっては、消防対策本部と署本部間の報告業務及び署本部の災害情報把握、活動状況の集約を主体とした運用とする。

  1. 防災指令が発令され、署本部が設置された際、「水防活動時の部隊運用」を元に指揮活動班が署本部編成表《様式３》を作成するので、消防対策本部に報告する。（全庁ファイル内の警防課が指定するフォルダーへ格納）
  2. 降雨状況、河川の水位状況によって警戒パトロールを実施した場合、《水防様式４》警戒パトロール状況、《水防様式５》水位観測表にパトロールコース、水位等の情報を入力し、消防対策本部へ報告（全庁ファイル内の警防課指定のフォルダーへ格納）する。
  3. 災害が発生すれば、指揮班が作成した復命書を元に事案管理シート（Teams）又はホワイトボード（災害状況管理表）に記入し、危機管理システムに事案入力する。併せて《水防様式７－２》災害状況管理表に事案を入力する。災害トリアージを実施した場合は、事案毎にトリアージした色を記入し、様式７－２にも記載する。
  4. 災害に出動した出動隊の災害概要報告を受け、ＯＡシステムに災害概要を入力する。完了すれば消防対策本部情報班に連絡する。入力内容を変更した場合も同様とする。
  5. 避難指示が必要となった場合、避難指示を発令する場所、建物棟数、世帯数、人数等の情報を署本部内で管理、共有する。
（３）災害トリアージ

トリアージの目的

  1. 管内の災害状況を把握し、署本部で追跡が必要な事案を明確化する。
  2. 管内の出動可能な部隊や転戦可能な部隊を把握するなかで、常駐警備隊が残り１隊の状況となれば続発対応を視野に入れ、災害内容を踏まえた出動体制とする。例えば、防災指令第２号では管内のポンプ隊５隊が増強され計10隊となるが、うち９隊が出動すれば水防出動の内容を精査し、内容により１線車での出動を一時見合わせる。重要な事案を最優先に対応するための事前対策である。
  3. 管制室が急報受信した事案は、管制室の状況（専用多数）によっては全て出動指令されるため、重複する事案が含まれることを念頭に置いた部隊運用を行う。

トリアージ区分及び条件

赤［高］

人的被害あり、又はその可能性がある。重要施設に被害あり、又はその可能性がある。

黄［中］

物的被害あり（緊急性あり）、又はその可能性がある。

緑［低］

上記以外。

※トリアージに関しては、別添「事案情報の流れ」を活用する。
【重要施設とは】情報通信・空港・鉄道・道路・電力・ガス・水道・行政・医療等の施設

指揮・活動班の対応

  • 出動可能な部隊数や転戦可能な部隊を把握、トリアージ赤［高］は最優先に対応
  • 事案処理：未処理事案がないか常に確認、未処理（至急・不急）及び処理済みBOXの有効活用（チェック表の活用）

続発時の対応（災害対応５件以上）

  • 赤［高］：臨時小隊、当務隊に対して早期の転戦指示
  • 黄［中］：臨時小隊、広報車又は事案終了小隊の転戦
  • 緑［低］：後刻での対応（事案保留）又は消防団対応
（４）部隊の編成

西消防署本部編成表《水防様式３》に基づく。増強部隊編成の都度に管制室（432又は433）へ電話連絡し、西消防署本部編成表を作成後、全庁ファイルサーバに保存、併せて作戦室（459）へ電話連絡をする。

  1. 警戒指令発令時：当務員及び係長を含む１名以上を配置し、管内警備に当たる。
  2. 防災指令第１号発令時：当務員及び参集者によって署本部各班及び西４を編成し、管内限定設定とする。
  3. 防災指令第２号発令時：
    • 当務員、日勤者（消防防災課員）及び参集者によって署本部各班の人員を増員し、本署で西４を編成、非番員参集者によって西１、西３、西15及び西５を各署所で編成する。
    • 西１、西３及び西15は、兼務解除及び管内限定設定とする。（兼務解除登録も同様に、署から電話連絡を受けた管制室が実施する。）
    • 西５で編成する小隊：救急要請等の状況によって救急隊を編成できるように努める／岩岡・神出地区の災害状況に応じて岩岡救急ステーションに配置する／特殊災害への対応（資機材搬送）ができること。
  4. 防災指令第３号発令以降：参集係員で広報車等を編成する。署本部の編成は防災指令第２号発令時を準用。
（５）巡視監視の留意事項

① 兵庫県地域別土砂災害危険度

２時間後に土砂災害警戒基準を超過すると予測されるエリア（１kmメッシュが黄色）又は１時間後に超過すると予測されるエリア（橙色）が管轄区域内に現れた場合にパトロールを実施する。パトロール後は、警戒パトロール状況（水防様式４）に記録する。

② 量水標の監視

  • 前川橋（伊川）：テレメータ・モニタリングカメラのいずれも設置されていないため、監視員を派遣する必要がある。
  • 山鳥橋：量水標は令和４年度に再設置。「兵庫県河川ライブカメラシステム」映像配信あり（枦谷川福谷局）。山鳥橋量水計は参考目安として、福谷及び二ツ屋橋量水標で補完する。
  • 上明石橋（明石川）：モニタリングカメラのみの設置であり、正確に水位を把握するには監視員を派遣する必要がある。
  • 消防対策本部への報告：情報班は基準水位の超過時、又は消防対策本部の求めに応じて、《水防様式５》水位観測表を全庁ファイルサーバに保存、作戦室（459）へ電話連絡をする。

③ 堤防巡視・監視（ため池）

  • 監視・巡視箇所は「防災重点ため池」の兵庫県防災工事等推進計画に指定されているため池15か所とする。
  • ため池管理者・ため池水利代表者から要請があったとき。
  • 記録的短時間大雨情報が発表されたとき（発表基準110mm/h）または、顕著な大雨に関する情報（線状降水帯等）が発表されたとき。
  • 神戸市地域防災計画では、ため池の監視は「ため池管理者・ため池水利代表者がパトロール等を行う」こととなっているが、上記のときは要請を待つことなく監視するものとする。

④〜⑧ その他の留意事項

  • 災害対応：マニュアル等資料１「水防活動時の部隊運用」に基づき実施する。
  • 出動車両：広域Ａモード時の水防指令に対しては、指揮班が選別する。
  • 使用無線機：増強部隊が使用する携帯無線機は、非常用携帯無線機とする。（別紙２－１西携帯無線配置一覧参照）
  • 当務隊優先の原則：当務部隊から優先的に水防活動にあてるものとし、増強部隊は続発災害に備えるものとする。水防事案以外の出動についても同様とする。
  • 長期間活動における部隊交代：指揮活動班は、現場活動が概ね３時間を超える（予想を含む）場合、現場交代を考慮する。現場交代は２交替又は３交替を基本としたローテーションを組み、交代決定時は次の事項を出動隊に周知・確認する。
    • 災害状況（災害概要、活動場所、危険情報等）
    • 気象状況（今後の降雨情報等の二次災害対策）
    • 活動状況（現在までの進捗と活動内容、活動時間等）
    • 体調管理（隊員の体調、事故発生時の連絡体制等）
（６）活動状況の報告及び集約
  1. 出動隊の決定：署本部（情報班）は、水防指令の出動隊を消防対策本部に報告する。
  2. 現場の状況：現場出動した隊は、現場対応後、水防情報集約シート復命書《水防様式６》を活用して災害現場の状況、被害の状況等を簡潔にまとめ、確実に署本部に伝達する。転戦する場合も無線、警防活動用スマートフォン、簡易AVM等を用いて署本部に連絡し、復命情報を伝達する。（復命情報は必ず解信をとるなど、情報共有に努めること。）
  3. 災害状況の入力報告：署本部（情報班）は、出動隊から報告された情報を消防OAシステムに入力し、完了すれば消防対策本部情報班に連絡する。入力内容を変更した場合も同様とする。
  4. 危機管理システムのアプリ「災害記録」の活用：現場写真や動画が署本部、消防対策本部、災害対策本部等と情報共有できるため、必要に応じて活用すること。危機管理システムのクロノロジーは、主として消防対策本部における市内の事案管理に活用するものであり、署本部で活用する場合は「水防時の消防OAシステム・危機管理システム入力マニュアル」を確認のうえ活用すること。
  5. 水防出動事案の管理：署本部（情報班）は、管内の水防出動事案について、災害状況管理表《水防様式７》を活用し事案管理を行うこと。Teams「西消防署防災指令関係」内にある事案管理シートにも入力を行い、署本部と消防対策本部、各出張所及び西区役所を始めとする区内防災機関と情報共有を行うこと。
（７）限定的地域における避難指示等発令の決定

限定的な地域における河川での浸水危険や土砂災害等に対して、避難指示等を発令する場合は、署本部で検討し、発令地域、時刻等の情報について消防対策本部と連携を図ったうえで署長が発令する。署本部は、各種気象情報や河川水位監視を確実に行い、時期を逸することなく発令する。

※避難指示等の発令は、神戸市地域防災計画風水害対策編の第５章「避難計画」に基づき発令。

次の事項が発生した場合、その初動対応についての検討は署長室で行う。

  • 避難指示の発令や解除
  • 行方不明者・死者の発生
  • 特に関係機関と連携強化する事案
（８）避難指示の判断

避難指示の主な判断基準は、気象警報・予測情報、河川基準水位及び現場の状況・実被害の３つを総合的に勘案して判断する。

① 避難指示等発令時期の判断基準

ア 土砂災害

  • レベル４土砂災害危険警報が発表され、兵庫県の「地域別土砂災害危険度」で、１時間後もしくは２時間後に土砂災害警戒基準を超過すると予測されるエリア（オレンジもしくは黄）が出現した場合
  • 現地確認により危険性が予測若しくは判断された場合。判断例：土砂移動現象、山鳴り、流木の流出、斜面の亀裂などの前兆現象を確認し、危険と判断し、より強く避難を促す必要がある場合

イ 外水氾濫（洪水）

  • 氾濫危険水位に到達すると予測される場合または到達した場合
  • レベル４大雨危険警報が発表され、気象庁の「洪水キキクル」で危険（紫）が出現した場合
  • 現地において災害発生の危険が高いと判断した場合

ウ 内水氾濫

  • 内水氾濫の発生が現地で確認され、被害が発生する蓋然性が高いと判断される場合には避難指示等を発令する。内水氾濫は予測が困難であるため、広報車等による注意喚起を積極的に行う。（別紙13－１参照）

エ ため池

消防職員及び消防団員は、現地確認の結果を署本部に報告する。署本部は、消防対策本部に報告し、危機管理局及び経済観光局農政計画課と情報共有、連携するものとする。ため池管理者からの緊急連絡系統は「神戸市地域防災計画」に基づく。

  • 防災重点ため池：兵庫県防災工事等推進計画に指定されているため池15箇所を現地確認の結果、水位が上昇し堤防決壊による被害が発生する蓋然性が高いと判断される場合は、浸水想定区域に避難指示を発令する。（別紙11－１参照）
  • ４号基準ため池：４号基準ため池11箇所を現地確認の結果、水位が上昇し堤防決壊による被害、もしくは土砂流入による被害が発生する蓋然性が高いと判断される場合は、ため池の浸水想定区域に避難指示を発令する。（別紙11－２参照）
  • その他の特定ため池：付近住民等から通報があった場合、現地確認を実施し被害が発生する蓋然性が高いと判断される場合は、浸水想定区域に対して避難指示を発令する。（別紙11－１参照）

② 発令範囲

ア 土砂災害

  • 該当区の土砂災害警戒区域
  • 現地確認により危険と判断した区域

イ 外水氾濫（洪水）

  • 氾濫危険水位に到達すると予測される場合または到達した場合、レベル４大雨危険警報＋「洪水キキクル」危険（紫）出現の場合 → 当該河川の計画規模降雨による洪水浸水想定区域に発令（別紙９参照）
  • 現地確認により危険と判断した地域：堤防決壊の予兆が認められる箇所又は決壊した箇所があるときは当該箇所が面する浸水想定区域に発令／一部閉鎖している橋梁があるときは当該橋梁が面する両岸の浸水想定区域に発令

ウ 内水氾濫

  • 内水氾濫の確認された場所が含まれる浸水想定区域に発令する。

③ 避難判断の資料

必要な情報

  • 避難世帯情報（世帯数と人員数）
  • 要救助者情報（行方不明者・負傷者）
  • 付近見取り図（避難区域がわかる地図。危機管理室と共有可能な図面）
  • 現場被害状況の写真
  • 避難先等の情報
  • 必要な様式

※口頭や無線での情報伝達には限界があるため、情報を図示して撮影するなど、（署本部等への）確実な情報送信に努めること。

現場活動隊の留意事項

  • 現場での退避は避難指示ではない。消防法第28条の消防警戒区域が該当。消防法第36条で水災以外の災害でも準用できるとされているため、土砂災害等による現場での退避は消防警戒区域の設定による退避となる。水災は水防法で警戒区域の設定が規定されている。
  • 避難指示は署長権限。現場の状況を署本部へ報告し、署本部で避難指示を発令すると判断すれば避難指示となる。

④ 避難指示等の解除時期

災害による直接の危険が無いと署長が判断した以下基準を満たす場合とする。

  • 量水標における水位が避難判断水位や橋桁下１ｍを下回り、かつ、大雨や洪水に関する警報が解除されて、今後の水位上昇がないと予測されるとき。
  • 土砂災害危険警報の発令が解除され、かつ、必要に応じて現地確認等を行い危険性が解消されたと予測されるとき。

⑤ 消防対策本部・区本部への提議

限定的地域を越え、一つの区レベル以上の広域に避難指示等を発令する必要があると判断した場合、その旨を消防対策本部・区本部に提議する。

（３）報告の留意事項

職員参集状況報告

職員配備数及び参集合計数は、当務員を含めた人数を入力する。職員配備数には、署本部編成表の警備人員欄の合計を入力する。危機管理システムが使用できない時は、警防規程様式第３（第59条関係）に従い報告する。

水位の状況報告

水位観測表《水防様式５》による記録は、各河川の量水標を現地で監視員が観測した場合に記録する。各河川で基準水位を超え避難指示等の発令を検討し始めた場合は、遅滞なく消防対策本部（情報班）に報告する。

水防作業の実施報告

消防職員、消防団員が出動して水防作業を実施した場合は、《水防様式13》にて報告する。水防作業とは、土嚢工法、排水ポンプによる排水等をいう。大規模な水防活動を実施した場合は、必要に応じて《水防様式13－２》により報告する。

局所的な地域に避難指示等を発令する場合

  • 発令の検討を始めたとき：署本部（情報班）は、消防対策本部（情報班）に概要を事前に連絡するとともに、区役所、警察等と連絡を密にする。
  • 発令を決定したとき：
    • 署本部（指揮活動班）は、消防対策本部（指揮班）へ電話連絡するとともに、避難情報発令状況《水防様式８》に必要事項を記入し、地図と合わせて全庁ファイルサーバに保存、もしくは作戦室にFAX（404）送信する。
    • 対象地域住民への指示は、速やかに現場出動隊が行い、避難世帯数、避難人員数、避難エリアを水利地図に記載し、署本部へ報告する。
    • 署本部においても、現場の状況が明確にわかるように、危機管理システム等を活用した情報共有を行うこと。
    • 西中池（押部谷町押部吉谷）の避難指示に関しては、神戸電鉄運転指令へ電話連絡を行うこと。（電話番号は部外秘のため紙の計画書を参照）
